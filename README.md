# ApplyMate AI

## AI Career Assistant

ApplyMate AI is an AI-powered career assistant designed to help users:

- Find Jobs
- Find Learnerships
- Apply for Universities
- Apply for TVET Colleges
- Generate ATS CVs
- Generate Cover Letters
- Prepare Interview Questions
- Track Applications
